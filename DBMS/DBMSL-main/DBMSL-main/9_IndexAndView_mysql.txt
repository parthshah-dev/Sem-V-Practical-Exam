i. Find the names of all instructors whose salary is greater than 25000.
ii. Update Instructor dept_name from CSE to IT.
iii. Create Simple Index on course table on the attribute dept_name and find out department-wise courses.
iv. Create a View to retrieve data of instructor with details of course, section, semester, and year he conducted lectures on.

Query Execution and Answers
i. Find all instructors whose salary > 25000
SELECT name FROM instructor
WHERE salary > 25000;

ii. Update Instructor dept_name from CSE to IT
UPDATE instructor
SET dept_name = 'IT'
WHERE dept_name = 'CSE';

iii. Create Simple Index on course.dept_name and find department-wise courses
CREATE INDEX idx_deptname ON course(dept_name);

SELECT dept_name, GROUP_CONCAT(title) AS courses
FROM course
GROUP BY dept_name;

iv. Create View to retrieve instructor-course details
CREATE VIEW instructor_course_view AS
SELECT i.name AS instructor_name, c.title AS course_title, t.sec_id, t.semester, t.year
FROM instructor i
JOIN teaches t ON i.T_ID = t.T_ID
JOIN course c ON t.course_id = c.course_id;

Theory -
What is MySQL
MySQL is an open-source relational database management system that stores structured data in tables and allows manipulation and querying using SQL commands.

Practical Related Concepts
1. Index:
An index improves query performance by allowing faster searches on specified columns.
Syntax:
CREATE INDEX index_name ON table_name(column_name);

2. View:
A view is a virtual table created from a query result. It simplifies queries and controls access.
Syntax:
CREATE VIEW view_name AS SELECT column1, column2 FROM table_name JOIN table2 ON condition;

3. SELECT with Condition:
Retrieve rows based on conditions such as comparison operators (>, <, >=, <=).

4. UPDATE:
Modifies existing rows in a table.
Syntax:
UPDATE table_name SET column=value WHERE condition;

5. GROUP BY & Aggregation:
Groups rows based on one or more columns. GROUP_CONCAT combines text values from multiple rows.
Syntax:
SELECT column, GROUP_CONCAT(column2) FROM table GROUP BY column;
